# かたつむり大作戦HP

## Project setup
```
npm install
```

### 開発時：HMR
```
npm run serve
```
下記よりアクセス可能。  
[http://localhost:8080]()

### Lints and fixes files
```
npm run lint
```

### 本番公開 (build & deploy)
```
npm run prod
```

### 本番用ビルド・docsディレクトリへ展開
```
npm run build
```

### docsディレクトリをGitHubPagesにデプロイする(gh-pagesブランチにpush)
```
npm run deploy
```
